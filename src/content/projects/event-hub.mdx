## Context & Background
Procurement Leaders hosted several large-scale events for procurement professionals. When I first joined the on-the-day content was being served to the delegates from a static websites. It shows the agends, speakers and sponsors. The pages were manually built for each event had no opportunity for interactivity.

## Implementation
- Allowed delegates to see other delegates, connect and chat to each other
- Allowed delegates to give feedback on sessions and speakers
- Automatically ingested content from Salesforce so updates to content were automatic
